GSCP
GSCP即全球社会责任合规方案，旨在制定一套可持续提高全球供应链的劳工标准及环境保护的社会责任最佳行为准则。为实现此目标，一些国际采购商创立了全球社会责任合规方案（GSCP），希望协作改善他们在很大程度上共享的供应源的（道德、社会和环境领域的）可持续性。为此，这些公司共同商讨制定一套全球通用、可持续提高劳工标准及环境保护的社会责任最佳行为准则。
